# Agent Spec — JobApplicationAgent

## Overview
JobApplicationAgent is a conversational AI agent built using ADK to help job seekers evaluate job fit and draft tailored cover letters. 
The agent operates in three distinct phases:
1. **Setup**: The candidate uploads their resume (PDF) or pastes their resume text. The agent extracts a structured profile (skills, projects, work history) and enriches it by fetching public GitHub repositories if a GitHub username is found.
2. **Analysis**: The user provides a job posting URL or description. The agent parses the posting, fetches details, runs a web search for company context, and calculates a transparent fit score across five dimensions (Technical skills, Experience level, Seniority, Domain fit, Culture fit).
3. **Output**: The agent generates a tailored cover letter draft that highlights relevant projects (e.g., Roadrunner) and addresses experience gaps (e.g., lack of domain experience) honestly. The user can refine the draft iteratively in plain language.

## Example Use Cases
### Scenario 1: Setup Candidate Profile
- **Input**: User uploads a resume PDF or text.
- **Output**: A structured summary of candidate details (name, title, skills, projects, public GitHub status) for user review and confirmation.

### Scenario 3: Batch Mode
- **Input**: Multiple job posting URLs.
- **Output**: Ranked comparison table with match scores and labels, with individual cover letter drafts available on demand.

## Tools Required
- **GitHub API (or scraper)**: To fetch public repository details for enriching candidate profiles.
- **Web Fetch/Scraper**: To fetch and clean job descriptions from various job boards/ATS systems (Greenhouse, Lever, Stripe, Block, etc.).
- **Web Search API**: To retrieve recent context/news about the target company.
- **Fit Scorer**: Component logic to calculate and combine matching criteria.

## Constraints & Safety Rules
- Never use generic placeholder phrases like "I am writing to apply..." or "I am passionate about..." in cover letters.
- Be honest and transparent about fit scores (do not inflate scores).
- Proactively address experience gaps in the cover letter instead of hiding or apologizing for them.
- Preserve candidate profiles across the session so they are only provided once.
- Refinement commands should update specific sections or styles, avoiding complete rewrites unless requested.

## Success Criteria
- Accurately parses candidate resumes and extracts key technical keywords.
- Successfully parses job boards and extracts key requirements.
- Calculates an explainable, multi-dimensional match score.
- Generates high-quality, professional, and evidence-supported cover letters that address skill/domain gaps.
